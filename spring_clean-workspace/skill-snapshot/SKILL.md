# Steps
1. Understand and explore the codebase.
2. Identify bugs and issues.
3. Propose solutions.
4. Identify core performance bottlenecks.
5. Propose solutions.
6. Synthesize and rank the issues from most critical to least.
7. Describe the complexity of each proposed solution.
